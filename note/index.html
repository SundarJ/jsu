What?

    When you click the +, you'll be given an opportunity to enter a title. This will create a note.

    Prepending the note title with a word followed by a colon (ie. 'foo:') will create a folder for that note, and any future notes with that prefix will also be added to that folder.

    Keyboard shortcuts:

    ?
    show this help
    ctrls
    export to notes.json
    ctrlo
    import from notes.json
    ctrlm
    create a new note
    ctrldel
    delete current note
